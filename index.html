Simple storage service for images.

ImageS3 is an image hosting service for developers. It is designed to store, resize and manage images for all your web and mobile apps in one place.

With ImageS3 you can create image plants to organize and store images for different apps, and create templates for resizing images into different pixels and percentages.

ImageS3 uses Amazon S3 to host image files and MongoDB to store image metadata.

ImageS3 provides you a web-based admin tool and your apps a set of REST APIs.

There is one more thing, ImageS3 is free and open.

Requirements

  • Java 7.
  • Amazon S3 account (http://aws.amazon.com/s3)
  • Play framework 2.3.x (https://www.playframework.com/documentation/2.3.x/Installing)
  • Latest MongoDB (http://docs.mongodb.org/manual/installation/)
