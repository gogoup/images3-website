ImageS3

ImageS3 is a free and open image hosting service for developers. It is designed to store, resize and manage images for all your web and mobile apps in one place.

Features

  • Create image plants to organize and store images on Amazon S3.
  • Design templates for resizing images into pixels or percents on demand.
  • REST APIs for easy application integration

  • Web-based admin tool with live monitoring:

Requirements

  • Java 7.
  • Play framework 2.3.x (https://www.playframework.com/documentation/2.3.x/Installing)
  • Latest MongoDB (http://docs.mongodb.org/manual/installation/)
  • Amazon S3 (http://aws.amazon.com/s3)

Installation

Download and unzip the compressed package:

$ curl -O https://github.com/gogoup/images3-dist/raw/master/images3-play-latest.zip
$ unzip images3-play-latest.zip

Update the following properties:

  • 'imagecontent.download.dir' in images3-play-/conf/imagecontent.properties.
  • 'image.processing.tempdir' in images3-play-/conf/imageprocessor.properties
  • 'mongodb.url', 'mongodb.port', 'mongodb.username' and 'mongodb.password' in images3-play-/conf/mongodb.properties

Enable execution persmissions:

$ chmod +x images3-play-<version>/bin/*

Running

$ ./images3-play-0.1.0/bin/images3-play

Open http://localhost:9000 in your browser, you will see the admin tool:

You can run ImageS3 on different port, for example 8080:

$ ./images3-play-0.1.0/bin/images3-play -Dhttp.port=8080
